# 메일 주문 프로그램

<!-- 이미지 -->

## 개요

네이버 스마트 스토어 제품 소개에 넣을 메일 주문 프로그램

## 환경

- 언어
  - JavaScript
- 라이브러리
  - React
  - BootStrap
  - Axios
  - Express
  - Heroku
  - nodemailer
  - dotenv
- 기타
  - jsdelivr

## 구현

- React, Bootstrap 활용 frontend 구성
- jsdelivr으로 woff 폰트 제공받음
- 사용자가 입력한 data을 backend로 전달
- data를 가공해 사업자 메일로 발송
